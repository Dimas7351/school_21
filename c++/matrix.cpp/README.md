## Description
C++ Library for working with matrices. With Unit tests.
## Build
      $ git clone https://github.com/ksilisk/matrix.cpp.git
      $ cd matrix.cpp
      $ make
## Tests
Unit tests with `gtest.h` C++ library.
      
      $ cd matrix.cpp
      $ make test
